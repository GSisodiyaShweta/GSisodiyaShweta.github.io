Shibu

Shibu is an Interactive ChatBot-Based Game.The game uses a sophisticated computational linguistics model to highlights subconsious gender biases in the language (English) of player.

Gender Inclusive Language(GIL) is a good social practice.But, the exisitng tools (seminars, workshops, videos etc) only serve as platform for introudction. Clearly, there is a gap between gap between the institutionalization and practical implementation of GIL.For effective implementation of GIL- individuals need to practise actively. Shibu is as a platform clearly underscore gap by highlighting the subconious biases in players langague (English).
